Capabilities

A tidy desktop, on autopilot.

01

Quiet by Default

Works in the background without stealing focus. A small toast tells you when something was filed away — click it to jump straight to the destination folder.

02

Flexible Rules

Match by extension or regex and route files to dynamic targets such as {year}/{month}.

03

Full History & Undo

Each move is recorded locally, and any of them can be rolled back with one click.

04

One-Click Cleanup

Years of accumulated clutter? Run a full scan and the whole folder gets sorted in seconds — no dragging, no decisions, no mess left behind.

05

Speaks Your Language

Picks up your system locale automatically, with six interface languages built in.

06

Local & Private

No accounts, no telemetry, no cloud sync. Everything lives in a small SQLite file on your own disk.

07

Featherweight Performance

A Rust + Tauri core keeps memory and CPU usage near zero, so you forget it is even running.

08

Safe With Active Downloads

A grace period plus file-lock checks guarantee nothing is touched until the download truly finishes.

Questions

Common questions,
honest answers.

How does it work?

The app watches the folders you pick (Downloads by default) from the system tray. When a new file lands, it waits a couple of seconds to be sure the transfer finished, then files it into the right subfolder according to your rules.

Does any data leave my machine?

Never. The app runs entirely offline — rules, history, and settings live in a local SQLite database and nothing is uploaded anywhere.

Can I define my own rules?

Yes — head to the Rules tab in Settings. Rules can match extensions or regex patterns and send files to destinations built from placeholders like {year}, {month}, or {filename}.

What if a file ends up in the wrong place?

Open History in Settings and hit undo — the file returns to exactly where it came from.

Does it launch with Windows?

By default, yes. Autostart can be switched off any time under Settings → General.

Is it really free?

Completely. It ships under the MIT License, so you are free to use it, fork it, and build your own version from source.

Is there a Dark Mode?

Yes — the theme follows your OS automatically, and you can pin Light, Dark, or System manually in Settings → General.

Will it slow my computer down?

No. Thanks to its Rust and Tauri foundation it uses barely any memory or CPU while idling in the background.
