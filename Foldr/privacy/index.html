The app

Foldr is a desktop application that runs entirely on your own machine. File names, file contents, and usage data never leave your device. Sorting rules, action history, and settings are held in a local SQLite database on your own disk.

The website

This site is a set of static pages. It runs no analytics and no tracking scripts, so no cookies are set and no visitor data is collected or stored.

A consent banner ships with the site for the case where you later add analytics. Any such tool must load only after a visitor explicitly accepts, and this section has to be rewritten to name the tool and list what it collects (typically IP address, device and browser details, pages visited, referrer, approximate location).

The only third-party request the pages make is to Google Fonts for the two webfonts. Your browser contacts Google's servers to fetch them, which exposes your IP address to Google. Self-host the font files to avoid that.

What we do not do

  • We do not sell, rent, or share data with third parties
  • We do not run advertising or tracking pixels
  • We do not embed social media widgets that follow you around
  • We do not build profiles of visitors

Your rights

Under GDPR you may request access to, correction of, deletion of, or restricted processing of your personal data. As things stand no personal data is collected, so there is nothing on file to act on — but the request channel below is open regardless.
