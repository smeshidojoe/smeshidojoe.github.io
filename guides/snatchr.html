Что это

Snatchr — универсальный загрузчик: скачивает что угодно одной командой.

Установка

  1. Установи Python 3.10+
  2. Склонируй репозиторий:
git clone https://github.com/smeshidojoe/Snatchr.git
cd Snatchr
pip install -r requirements.txt

Первый запуск

Пример команды:

python snatchr.py <ссылка>
